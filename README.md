## How to use Template Generator?

  
1. Install Template Generator

  

```

npx cli-template-self

```

  

2. Create a new project


```

<project-name>

```

  

3. Choose a template

	  [`more`](Templates)

  
4. Start the project

  
```

cd <project-name>
pnpm i
pnpm dev

```
